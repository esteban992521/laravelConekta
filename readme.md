## LaravelConekta

LaravelConekta es un ejemplo sencillo de integracion de la pasarela de pagos Conekta, solo ingresa en el archivo .env  tus credenciales.

- CONEKTA_SANDBOX_API_KEY_SECRET
- CONEKTA_SANDBOX_API_KEY_PUBLIC
